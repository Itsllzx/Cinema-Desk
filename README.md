# 🎬 دفتر سینما – یه دفترچه خاطرات آفلاین برای فیلم‌هات

سلام!  
این یه اپ وب ساده‌ست که باهاش می‌تونی هر فیلمی دیدی رو توی یه دفترچه شخصی ثبت کنی.  
همه چی توی همون مرورگر گوشی یا لپ‌تاپته، اینترنت نمی‌خواد، سرور نمی‌خواد، فقط یه فایل HTML.

---

## چه چیزایی توش هست؟

- ➕ ثبت فیلم با اسم، سال، ژانر، امتیاز ستاره‌ای (۱ تا ۵)، لایک، عکس (اختیاری) و یه جای خالی برای نوشتن حس و حال خودت موقع دیدن فیلم  
- 📔 ژورنال شخصی: مثلاً «وقتی این فیلم رو دیدم یاد فلان روز افتادم...»  
- 📀 ساخت پلی‌لیست دلخواه: مثلاً «فیلم‌هایی که باید دوباره ببینم» یا «بهترین‌های سال»  
- 🎁 یه دکمه سورپرایز که از بین فیلم‌های ۵ ستاره‌ات یکی رو بهت پیشنهاد میده دوباره ببینی  
- 📤 خروجی گرفتن از همه فیلم‌ها و پلی‌لیست‌ها (یه فایل JSON ساده) و دوباره وارد کردنش برای بکاپ یا اشتراک  
- 🔍 فیلتر بر اساس ژانر، لایک‌ها، و یه فیلتر ترکیبی  
- 🖼️ کلیک روی هر فیلم، تمام جزییاتش رو بهت نشون میده (مخصوصاً یادداشت بلندت)  
- 📱 تو گوشی و ویندوز و مک کار می‌کنه، ریسپانسیوه، تم دارک با رنگ طلایی-نارنجی، چشم نوازه  

---

## چجوری کار می‌کنه؟

1. فایل `index.html` رو دانلود کن (یا خودت بسازش).
2. روش دابل کلیک کن تا تو مرورگر باز بشه.
3. هر چی فیلم داری رو ثبت کن، براش عکس بذار، امتیاز بده، براش خاطره بنویس.
4. همه چی خودکار توی گوشی یا کامپیوترت ذخیره می‌شه (همون localStorage مرورگر).
5. هر وقت خواستی می‌تونی یه فایل بکاپ بگیری و بندازی تو گوگل درایو یا برا دوستت بفرستی.

> هیچ اینترنتی لازم نیست، هیچ نصب و سروری هم نداره.

---

## با چی ساخته شده؟

- HTML و CSS خالص
- جاوااسکریپت وانیلا (بدون فریم‌ورک)
- ذخیره‌سازی با localStorage
- طراحی با flex و grid و backdrop-filter

---

## چطوری تو گیتهاب کمک کنم؟

- فورک کن
- یه برنچ جدید بزن
- تغییراتتو اعمال کن
- پول ریکوئست بده

هر مشکلی یا پیشنهادی هم توی بخش Issues بنویس.

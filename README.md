# 🎬 Cinema Notebook – Offline Movie Journal

> **A minimalist, personal, offline movie diary**  
> No server, no internet – just a single HTML file running on your phone or laptop.

## ✨ Features

- ✅ **Add movies** with title, year, genre, star rating (1-5), like, personal journal note, and even upload a poster image (Base64)
- 📔 **Emotional journal** – write down how you felt while watching, what memories came to mind
- 📀 **Custom playlists** (e.g., "Watch Later", "All-Time Favorites", "Friends' Picks") – add any movie to any list
- 🎁 **Random surprise pick** – suggests one of your 5-star movies to rewatch
- 📤 **Full Export / Import** – JSON backup of all movies & playlists, shareable and restorable
- 🔍 **Smart filters** by genre, likes, and year
- 🖼️ **Full movie details** – click on any card to see the complete journal entry, edit or delete
- 📱 **Fully responsive** – works great on Windows, Android, desktop
- 🌙 **Dark minimal theme** with a warm gold–midnight palette – elegant and easy on the eyes

---

## 🚀 How to use (no installation)

1. Download the `index.html` file from this repository (or create it yourself).
2. Open it in any browser on your phone (Chrome, Samsung Internet, Firefox) or laptop.
3. Everything works offline – data is saved to `localStorage`.
4. For better compatibility on **Android** with `file://`, use browsers like `Via` or `Brave`.  
   (Chrome also works fine.)

> **No server, no internet, no installation required.**

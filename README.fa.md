# 🎬 Cinema Office – دفتر سینما

> **یک دفترچه خاطرات سینمایی آفلاین، شخصی و مینیمال**  
> بدون نیاز به سرور، بدون اینترنت، فقط با یک فایل HTML در گوشی یا لپ‌تاپ شما.

## ✨ ویژگی‌ها

- ✅ **ثبت فیلم** با نام، سال، ژانر، امتیاز ستاره‌ای (۱ تا ۵)، لایک، یادداشت ژورنال شخصی و حتی آپلود عکس (Base64)
- 📔 **ژورنال عاطفی** – بنویس هنگام دیدن آن فیلم چه حسی داشتی، چه خاطره‌ای برات تداعی شد
- 📀 **پلی‌لیست‌های دلخواه** (مثل «ببینم»، «بهترین‌ها»، «پیشنهاد دوستان») – فیلم‌ها را به هر لیستی اضافه کن
- 🎁 **پیشنهاد تصادفی** – از میان فیلم‌های ۵ ستاره‌ات یکی را برای تماشای دوباره پیشنهاد می‌دهد
- 📤 **Export / Import کامل** – خروجی JSON از همه فیلم‌ها و پلی‌لیست‌ها و بازیابی مجدد (برای اشتراک یا بکاپ)
- 🔍 **فیلتر هوشمند** بر اساس ژانر، لایک‌ها، و ترکیب با سال
- 🖼️ **نمایش جزئیات کامل فیلم** با کلیک روی کارت، قابل ویرایش و حذف
- 📱 **واکنش‌گرا (Responsive)** – عالی روی ویندوز، اندروید، و دسکتاپ
- 🌙 **تم دارک مینیمال با پالت طلایی–نیمه‌شب** – زیبا و چشمنواز

---

## 🚀 نحوه استفاده (بدون هیچ نصب‌ی)

1. فایل `index.html` را از ریپازیتوری دانلود کن (یا خودت بساز).
2. آن را در مرورگر گوشی (کروم، سامسونگ اینترنت، فایرفاکس) یا لپ‌تاپ باز کن.
3. همه‌چیز آفلاین کار می‌کند – داده‌ها در `localStorage` ذخیره می‌شوند.
4. برای اجرای بهتر روی **اندروید** می‌توانی از مرورگر `Via` یا `Brave` استفاده کنی.  
   (در کروم فایل‌های محلی با `file://` مشکلی ندارند.)

> **نیازی به سرور نیست، نیازی به اینترنت نیست، نیازی به نصب نیست.**

---

## 🧰 تکنولوژی‌ها

- HTML5 / CSS3 / JavaScript خالص (Vanilla JS)
- LocalStorage برای ذخیره‌سازی
- IndexedDB پشتیبانی نمی‌شود (برای حداکثر سازگاری با `file://` از localStorage استفاده شده)
- طراحی با CSS Grid/Flex، backdrop-filter و متغیرهای سفارشی
